# Career Project

Проект Node.js для карьерной программы

## Структура проекта

```
├── config.js      # Конфигурация приложения (appName, настройки)
├── logger.js      # Обёртка для логгера с использованием config.appName
├── scheduler.js   # Планировщик периодических задач
└── package.json
```

## Начало работы

```bash
npm install
npm start
```

## Модули

- **config.js** — хранит appName и настройки проекта.
- **logger.js** — обёртка для console.log, выводит отформатированные сообщения с именем приложения и временной меткой.
- **scheduler.js** — инициализируется синхронно (логирует запуск), предоставляет метод scheduleTask(name, interval, task) для выполнения периодических задач. Включает встроенную задачу heartbeat, которая логирует "running" каждые 10 секунд.
